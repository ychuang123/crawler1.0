1. 浏览网页
    浏览器模式。在浏览网页的过程中，根据需要，可随时采集目标数据
  2. 规则库
    定义要采集数据的抽取规则
  3. 程序库
    定义目标数据的自动采集流程
  4. 任务库
    定义数据采集的运行任务
    对于个性化的数据采集，先配置规则和程序，再配置任务
    对于新闻采集，可以选择自动新闻采集。不用事先定义规则和程序，简单高效。采集结果可能有噪音
  5. 验证码
    解决自动采集过程中出现的验证码
  6. 系统设置
    设置系统参数
  7. 软件授权
    未授权用户只能对最前面的 3 个任务采集数据，每个任务最多只能抓取1000条数据
    对于未授权用户，可以通过删除前面的任务，实现对后面任务的采集
    对于未授权用户，任务采集的数据达到1000条记录后，可以删除采集结果数据，从而继续采集
    软件有效期到期后，当前版本将不能使用。请提前下载新版本
