定义程序的运行，包括启动方式、运行需要的种子URL等参数
  1. 点击 增加任务 按钮，弹出 选择程序 对话框
  2. 点击 程序 下拉选择框，选择新浪新闻详情2
  3. 点击 确定 按钮，弹出 增加任务 对话框
  4. 输入任务名称：新浪新闻详情2
  5. 点击 运行类型 下拉选择框，选择手动运行
  6. 点击 文件 按钮，弹出文件选择对话框，选择种子文件
    选择的文件内容必须是 ANSI 编码格式的
    种子文件的每一行是要采集的新闻详情页URL，比如：
    https://news.sina.com.cn/c/2018-11-10/doc-ihnstwwq1955194.shtml
    https://mil.news.sina.com.cn/jssd/2018-11-10/doc-ihmutuea8891565.shtml
    https://mil.news.sina.com.cn/jssd/2018-11-10/doc-ihmutuea8887424.shtml
  7. 点击 确定 按钮提交
