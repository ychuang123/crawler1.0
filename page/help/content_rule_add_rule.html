定义规则
  1. 点击 增加规则 按钮，弹出规则类型选择对话框
  2. 点击 规则类型 下拉选择框，选择规则类型
    数据抽取规则：从网页抽取结构化数据记录
    网页交互操作：与网页进行交互操作，在网页上输入参数，然后点击按钮。比如翻页、登录、基于 JavaScript 的翻页跳转
    验证码识别：从网页识别验证码图片
  3. 点击 网页数量 下拉选择框，选择学习网页数量
    单网页：指定 1 个网页样本学习规则
    双网页：同时指定 2 个网页样本学习规则
    首次选择单网页。当学习到的规则测试时总是采集失败，这时考虑采用双网页来学习
  4. 点击 确定 按钮，进入浏览网页模式
  5. 在地址栏中输入网址，回车
  6. 等网页加载完成后，点击 加载完成 按钮，进入规则定义模式
