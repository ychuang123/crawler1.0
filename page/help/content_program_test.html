输入运行程序需要的数据，测试程序。如果程序有网页交互规则，且交互规则有输入参数，也需要指定
  1. 点击 测试 按钮，弹出 输入参数 对话框
  2. 输入种子URL，或者点击 文件 按钮选择种子文件
    选择的文件内容必须是 ANSI 编码格式的
    如果种子URL只有1个，直接输入种子URL，否则，须将种子URL写到本地文件，一行一个，然后选择该文件
    种子URL格式1：cmd,arg:url
    命令及参数，可以有多个，最后面为网页URL
    命令与参数使用逗号分隔，命令之间，以及命令与网页URL之间使用冒号分隔
    种子URL格式2：cmd,arg:@file:file
    命令及参数，可以有多个，最后面的 @file:file 表示从本地文件 file 读取
    命令与参数使用逗号分隔，命令之间，以及命令与@file之间使用冒号分隔
    命令1：@file
    从本地文件加载种子URL。没有参数，在所有命令之后，冒号后面为本地文件
    @file:D:\seed.txt
    命令2：@tag
    给种子URL分配标签。有 1 个参数：标签
    @tag:时政:http://shizhen.sina.com.cn
    @tag:军事:http://junshi.sina.com.cn
    命令3：@rep
    批量生成多个种子URL。有 4 个参数：要替换的占位符、循环起始数、循环结束数、步长
    @rep,%d,1,31,1:http://news.sina.com.cn/md_%d
    上面的种子 URL 经过 @rep 命令替换，生成如下 31 个种子URL：
    http://news.sina.com.cn/md_1
    http://news.sina.com.cn/md_2
    命令4：@input
    为网页交互规则指定参数
    @input后面的参数为该种子下的步骤序号，交互规则名，以及规则的输入值
    @input,1,关键词搜索,雾霾:http://baidu.com
  3. 如果程序有网页交互规则，则需要指定交互规则的参数值，或者点击 文件 按钮选择参数文件
    选择的文件内容必须是 ANSI 编码格式的
    多个参数值之间使用逗号分隔
    如果有多行参数值，须将参数值写到本地文件，然后选择该文件
    如果上面第 2 步种子URL中已经指定了规则的输入值，这里可以填 @seed 表示从种子URL中取
    如果有多行参数值，每次只会依次使用其中一行。比如登录账户和密码
    如果每行参数值都要采集，须使用上面第 2 步 input 指令将多行参数值配置到种子URL中
  4. 点击 确定 按钮，进入程序测试运行模式
