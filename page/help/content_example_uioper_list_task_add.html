定义程序的运行，包括启动方式、运行需要的种子URL等参数
本例假定要搜索并采集下面关键词的新闻：
联合国 人权
转基因
南海
  1. 点击 增加任务 按钮，弹出 选择程序 对话框
  2. 点击 程序 下拉选择框，选择百度新闻搜索
  3. 点击 确定 按钮，弹出 增加任务 对话框
  4. 输入任务名称：百度新闻搜索
  5. 点击 运行类型 下拉选择框，选择手动运行
  6. 点击 文件 按钮，弹出本地文件选择对话框，选择种子URL文件
    选择的文件内容必须是 ANSI 编码格式的
    每一行为一个种子URL。使用 input 指定搜索关键词
  7. 输入 @seed，从种子URL获取关键词
  8. 点击 确定 按钮提交
